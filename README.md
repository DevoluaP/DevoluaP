## Desenvolvedor Full Stack Júnior | Junior Full Stack Developer

##### PT
Tenho experiência em front-end utilizando HTML, CSS, JavaScript, React.js, Angular, TypeScript e React Native. No back-end, trabalho com Node.js, PHP, Python, Java e APIs RESTful, integrando sistemas e gerenciando bancos de dados SQL (MySQL e PostgreSQL) e NoSQL (MongoDB). ##### EN
